## 简介

购物商城后端项目是一个基于Spring Boot 3.2.5构建的现代化电商系统后端服务。项目采用分层架构设计，集成了JWT认证、MyBatis-Plus ORM框架、MySQL数据库、Knife4j接口文档等核心技术组件。系统提供了完整的用户认证、商品管理、订单处理、购物车等功能模块。

## 环境准备

### JDK 17 安装配置

项目要求使用Java 17版本，这是Spring Boot 3.x的最低要求。

**安装步骤：**
1. 访问Oracle官网或OpenJDK官网下载JDK 17
2. 解压到指定目录（如 `/usr/lib/jvm/java-17-openjdk`）
3. 配置环境变量：
   ```bash
   export JAVA_HOME=/usr/lib/jvm/java-17-openjdk
   export PATH=$PATH:$JAVA_HOME/bin
   ```
4. 验证安装：
   ```bash
   java -version
   javac -version
   ```

### MySQL 数据库安装

项目使用MySQL作为数据存储，建议版本为8.0以上。

**安装步骤：**
1. 下载MySQL Community Server
2. 按照官方安装向导完成安装
3. 启动MySQL服务
4. 创建数据库和用户：
   ```sql
   CREATE DATABASE appdb CHARACTER SET utf8mb4 COLLATE utf8mb4_general_ci;
   CREATE USER 'appuser'@'localhost' IDENTIFIED BY 'AppPassw0rd!';
   GRANT ALL PRIVILEGES ON appdb.* TO 'appuser'@'localhost';
   FLUSH PRIVILEGES;
   ```

### Maven 安装配置

项目使用Maven 3.6+进行依赖管理和构建。

**安装步骤：**
1. 下载Apache Maven
2. 解压到指定目录
3. 配置环境变量：
   ```bash
   export MAVEN_HOME=/usr/local/apache-maven
   export PATH=$PATH:$MAVEN_HOME/bin
   ```
4. 验证安装：
   ```bash
   mvn -version
   ```

## 数据库初始化

项目提供了完整的数据库初始化脚本，包含表结构定义和测试数据。

### 数据库连接配置

在 `application.yml` 中配置数据库连接信息：

```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/appdb?useUnicode=true&characterEncoding=utf8&useSSL=false&serverTimezone=Asia/Shanghai&allowPublicKeyRetrieval=true
    username: root
    password: RootPassw0rd!
    driver-class-name: com.mysql.cj.jdbc.Driver
```

### 执行初始化脚本

**步骤1：执行Schema脚本**
```bash
mysql -u root -p appdb < src/main/resources/db/schema.sql
```

**步骤2：执行数据脚本**
```bash
mysql -u root -p appdb < src/main/resources/db/data.sql
```

**步骤3：验证数据库结构**
```sql
SHOW TABLES;
DESCRIBE tb_user;
DESCRIBE tb_product;
```

### 图片数据导入（可选）

如果需要完整的测试数据，包括图片资源：

```bash
python3 src/main/resources/db/upload_images.py
mysql -u root -p appdb < src/main/resources/db/import_with_images.sql
```

## 常见问题排查

### 数据库连接问题

**问题：无法连接数据库**
```bash
# 检查数据库服务状态
sudo systemctl status mysql

# 测试数据库连接
mysql -h localhost -u root -p

# 检查防火墙设置
sudo ufw status
```

**解决方案：**
1. 确认MySQL服务已启动
2. 检查用户名和密码配置
3. 验证网络连接和端口开放

### JWT认证问题

**问题：登录失败或令牌无效**
```bash
# 检查JWT配置
cat src/main/resources/application.yml | grep jwt

# 验证密钥长度
echo "qoder-mall-jwt-secret-key-2024-spring-boot" | wc -c
```

**解决方案：**
1. 确认JWT密钥配置正确
2. 检查令牌过期时间设置
3. 验证BCrypt密码编码

### 端口冲突问题

**问题：端口8080被占用**
```bash
# 查找占用端口的进程
lsof -i :8080
netstat -an | grep :8080

# 修改端口配置
# 在application.yml中修改server.port
```

### 依赖问题

**问题：Maven依赖下载失败**

```bash
# 清理并重新下载依赖
mvn clean install -U

# 配置Maven镜像源
```
